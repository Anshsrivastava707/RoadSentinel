Technical Methodology

Transparent explanation of data sources, predictive models, and validation protocols.

1. Data Sources

The system integrates multi-modal data from authoritative sources across India:

  • NHAI & State PWD Logs: Historical repair records (2018–present).
  • Traffic Monitoring Systems: Vehicle count data from NTC.
  • Environmental Sensors: Rainfall, temperature from IMD stations.
  • Citizen Reports: Verified submissions via UMANG app.
  • Geospatial Imagery: Satellite and street-level imagery.

2. Predictive Modeling Framework

We employ a hybrid machine learning approach:

1

Feature Engineering

42 features derived from raw data including traffic density, monsoon exposure, soil type, road age, and historical recurrence patterns.

2

Ensemble Modeling

Gradient Boosting (XGBoost) + Random Forest trained on 5.2M+ historical records.

3

Priority Scoring

Composite index weighted by: structural risk (40%), public impact (30%), economic cost (20%), strategic importance (10%).
